# azure-resources

### ConexaoStorageAccount.py

Nesse notebook criamos uma conexão com um recurso Azure e utilizamos isso para analisar, manipular e consumir os arquivos que estão lá dentro. 

  #### Dependências
  - azure.storage.blob (pip)
  - python 3.10
